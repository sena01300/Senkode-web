MENU REFORM

メニュー改善：28品から12品へ

削ることは、弱めることではない。「選ぶ理由」を際立たせることだ。

なぜ削るのか
28品のメニューはお客様を迷わせる。心理学の「選択のパラドックス」—選択肢が多いほど、人は決められず不満を感じやすくなる。 マクドナルドが3種類のバーガーから始まったのは偶然ではない。

①Troisでしか食べられない一品（シグネチャー）、 ②食材原価率30%以下で利益が出るもの、 ③ランチとディナーで共用できるもの。この3基準で12品まで絞る。

04 / 08
MENU COPYWRITING

メニュー名の書き直し

「豚バラ煮込み¥2,200」と「7時間煮込んだ豚バラ、赤ワインソース¥2,200」——同じ料理でも、伝わるものが変わる。

MAIN DISH — シグネチャー
鴨胸肉のロースト オレンジソース
→
鴨のコンフィ、10年間変わらないソース。
— Bistro Troisの看板料理 —
「ロースト」は調理法。お客様には関係ない。「10年変わらない」という一言で、職人の誇りとリピーターの期待が同時に伝わる。副題でシグネチャーであることを明示し、注文の背中を押す。
MAIN DISH — ストーリー
豚バラ肉の煮込み
→
7時間煮込んだ豚バラ、
赤ワインソース。
「煮込み」では何の価値も伝わらない。「7時間」という数字が、手間と覚悟を可視化する。お客様は料理を食べるとき「これに7時間かけたんだ」と感じながら食べる。同じ料理が別物になる。
DESSERT — SNS拡散設計
ガトーショコラ
→
割れたガトーショコラ
（わざと割ってます）
「なぜ割れているのか」という疑問がSNS投稿のきっかけになる。「わざと割ってます」というカッコ書きが笑いと共感を生む。お客様が自然に写真を撮り、キャプションを考え、投稿する——この流れを設計している。
