Problem — 従来の経費精算

3つの無駄が、経理の時間を奪っている

月次締めのたびに繰り返される非効率。従業員50〜100名規模では、これが年間で数百万円の機会損失になっています。

01
🗒️

紙レシートの紛失・劣化

出張・外食・消耗品購入のたびにレシートを保管する運用は破綻しやすい。 月末になって「見つからない」「読めない」が頻発し、差し戻しや再申請で二重の手間が発生します。

月平均 3〜5件の紛失
02
⌨️

手入力ミスと転記エラー

金額・日付・勘定科目を手打ちするたびに人的ミスが生まれます。 ゼロの数が一つ違うだけで申請却下、修正対応、再承認という ループが経理担当者を消耗させます。

転記ミス率 平均4.7%
03
⏳

承認ルートの非効率

紙の申請書が担当者のデスクで止まり、上長が不在なら翌週まで待機。 承認が遅れれば締め作業も遅延し、月次決算スケジュール全体を 圧迫します。

平均承認リードタイム 4.2日
Solution — ExpenseBotの仕組み

撮影から記録まで、すべて自動

4つのステップで経費申請が完結。従業員はスマホのカメラを向けるだけ、経理担当者は確認ボタンを押すだけです。

📱
Step 01 — Capture
スマホでレシート撮影

専用Google FormまたはLINE Botから画像をアップロード。 特別なアプリのインストール不要。普段使いのスマホだけで完結します。 撮影と同時に日時・位置情報も自動付与されます。

⚙️
Step 02 — Extract
Claude Vision APIが自動読み取り

Google Apps Scriptが画像を受け取り、Claude Vision APIへ転送。 AIが金額・店名・日付・消費税区分を高精度で抽出します。 手書きレシートや傾いた撮影にも対応。

🗂️
Step 03 — Classify
勘定科目・カテゴリを自動分類

抽出データを交通費・接待費・消耗品費などの勘定科目に自動振り分け。 企業ごとの分類ルールをプロンプトに組み込むことで、 既存の会計ルールに準拠した仕訳が可能です。

✅
Step 04 — Approve
スプレッドシート記録 + 承認通知

分類済みデータをGoogleスプレッドシートへ自動書き込み。 上長へはGmailまたはSlackで即時通知が届き、 ワンクリック承認が可能。リアルタイムで経費集計が完了します。

Code — 実装コアスニペット

GAS × Claude Vision API実装例

Google Apps Script からClaude Vision APIを呼び出し、レシート情報をJSON形式で取得する中核処理です。

ExpenseBot.gs — analyzeReceipt()
// ============================================================ // ExpenseBot — GAS × Claude Vision API コアスニペット // Google Apps Script で動作 // ============================================================ const CONFIG = { CLAUDE_API_KEY: PropertiesService.getScriptProperties().getProperty('CLAUDE_API_KEY'), API_URL: 'https://api.anthropic.com/v1/messages', MODEL: 'claude-opus-4-5', SHEET_NAME: '経費記録', }; /** * レシート画像URLを受け取り、Claude Vision APIで解析して * 金額・店名・カテゴリをスプレッドシートに書き込む */ function analyzeReceipt(imageUrl, submittedBy) { const imageBase64 = fetchImageAsBase64(imageUrl); const payload = { model: CONFIG.MODEL, max_tokens: 512, messages: [{ role: 'user', content: [ { type: 'image', source: { type: 'base64', media_type: 'image/jpeg', data: imageBase64, }, }, { type: 'text', text: `このレシート画像を解析し、以下のJSONのみを返してください。 { "store_name": "店名", "date": "YYYY-MM-DD", "total_amount": 数値, "tax_amount": 数値, "category": "交通費|接待費|消耗品費|その他", "items": [{"name": "品目", "amount": 数値}], "notes": "特記事項があれば" }`, }, ], }], }; const response = UrlFetchApp.fetch(CONFIG.API_URL, { method: 'post', contentType: 'application/json', headers: { 'x-api-key': CONFIG.CLAUDE_API_KEY, 'anthropic-version': '2023-06-01', }, payload: JSON.stringify(payload), }); const result = JSON.parse(response.getContentText()); const data = JSON.parse(result.content[0].text); writeToSheet(data, submittedBy); sendApprovalNotification(data, submittedBy); return data; } /** スプレッドシートへ書き込み */ function writeToSheet(data, submittedBy) { const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(CONFIG.SHEET_NAME); sheet.appendRow([ new Date(), submittedBy, data.date, data.store_name, data.category, data.total_amount, data.tax_amount, '未承認', data.notes, ]); } /** 承認者へのメール通知 */ function sendApprovalNotification(data, submittedBy) { const approverEmail = PropertiesService.getScriptProperties() .getProperty('APPROVER_EMAIL'); GmailApp.sendEmail(approverEmail, `[承認依頼] ${data.store_name} ¥${data.total_amount}`, `${submittedBy} から経費申請が届きました。\n` + `金額: ¥${data.total_amount}\n科目: ${data.category}\n` + `スプレッドシートで承認してください。` ); }
ExpenseBot.gs — fetchImageAsBase64()
/** Google DriveのファイルIDからBase64画像を取得 */ function fetchImageAsBase64(fileId) { const file = DriveApp.getFileById(fileId); const blob = file.getBlob(); return Utilities.base64Encode(blob.getBytes()); } /** Google Form 送信時トリガーで自動実行 */ function onFormSubmit(e) { const responses = e.namedValues; const imageUrl = responses['レシート画像'][0]; const submittedBy = responses['お名前'][0]; // Google Form の画像URLからファイルIDを抽出 const fileId = imageUrl.match(/[-\w]{25,}/)?.[0]; if (!fileId) return; analyzeReceipt(fileId, submittedBy); }
