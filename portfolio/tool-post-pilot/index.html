Architecture / How It Works

仕組み

月初に一度フォームを入力するだけ。あとはGASとClaude APIが全工程を自動処理し、スプレッドシートに整理された投稿文が届く。

📋
Step 01 / Input
月次入力フォーム（Google Forms）

業種・ターゲット・今月のテーマ・NGワード・投稿頻度を入力。1回の入力で30日分のコンテンツ生成がスタートする。月初トリガーで自動実行も設定可能。

⚙️
Step 02 / Calendar
GAS — コンテンツカレンダー自動生成

月初トリガーが起動し、Google Apps Scriptが月全体のカレンダーを自動構築。曜日・時期・イベントを考慮したコンテンツ配分を自動で設計する。

🤖
Step 03 / Generation
Claude API — 投稿文・ハッシュタグ・投稿時間を生成

各投稿のブリーフをClaude APIに送信。投稿本文・ハッシュタグ・最適投稿時間・エンゲージメントTipsをJSON形式で返す。X・Instagram・LinkedIn向けに自動で最適化。

📊
Step 04 / Output
Googleスプレッドシート — 一覧整理・保存

投稿文・プラットフォーム・日時・ハッシュタグ・Tipsを自動でスプレッドシートに整理保存。担当者はシートを開いて内容をレビューするだけでよい。

✅
Step 05 / Publish
担当者レビュー → 承認 → スケジュール投稿

最終チェックと微調整は担当者が行う。承認後は各プラットフォームのスケジュール機能を使って予約投稿するだけ。月30投稿の運用が週30分以内で完結する。

// PostPilot — 月次コンテンツ生成メイン関数 function generateMonthlyContent() { const config = getConfigFromSheet(); const calendar = buildContentCalendar(config); calendar.forEach((day, index) => { Utilities.sleep(500); // APIレート制限対応 const posts = generatePostsForDay(day, config); writePostsToSheet(posts, index); }); sendSummaryEmail(config.notifyEmail, calendar.length); } function generatePostsForDay(day, config) { const platforms = ['X', 'Instagram', 'LinkedIn']; const results = {}; platforms.forEach(platform => { if (!config.platforms.includes(platform)) return; const prompt = buildPostPrompt(day.theme, platform, config); const response = callClaudeAPI(prompt); results[platform] = JSON.parse(response); }); return results; } function buildPostPrompt(theme, platform, config) { const platformGuide = { 'X': '140字以内・ハッシュタグ2個以下・改行を効果的に使う', 'Instagram': '2200字以内・感情に訴える・ハッシュタグ10〜15個', 'LinkedIn': '1300字以内・専門的・ビジネス価値を明確に' }; return ` 業種: ${config.industry} ターゲット: ${config.targetAudience} 今月のテーマ: ${theme} ブランドトーン: ${config.brandTone} NGワード: ${config.ngWords.join(', ')} ${platform}向けの投稿文を作成してください。 ガイドライン: ${platformGuide[platform]} JSON形式で返してください: { "text": "投稿本文", "hashtags": ["タグ1", "タグ2"], "best_time": "HH:MM", "engagement_tip": "エンゲージメントを上げるコツ" }`; }
