Architecture / How It Works

処理フロー

📝
Step 01 / Input
営業担当者 — 打ち合わせメモをGoogleフォームに入力

顧客名・案件名・打ち合わせメモ（自由記述）・概算予算・希望納期を入力。スマートフォンからもアクセス可能。フォーム送信と同時に処理が自動起動する。

⚙️
Step 02 / Trigger
GAS — フォーム送信トリガー起動・項目の整理

Google Apps Scriptがフォーム送信イベントを検知。入力データを取得してテキスト整形を実行。カテゴリ分類と構造化を行ってからClaudeへ送信する。

🤖
Step 03 / AI Generation
Claude API — 見積項目の展開・単価補完・文章生成（JSON返却）

claude-haiku-4-5がメモから見積項目を自動展開。業種DBに基づいて単価を補完し、合計金額・支払条件・挨拶文をJSON形式で返却する。1件あたり平均20秒。

📊
Step 04 / Output
Googleスプレッドシート — 見積書テンプレートに自動入力

事前に設定した見積書テンプレートシートへ、取得したJSONデータを自動で書き込む。顧客名・明細・合計・挨拶文がすべて整った状態で出力される。

📤
Step 05 / Delivery
PDF出力 → 担当者へメール通知

スプレッドシートをPDF化してGoogleドライブに保存。完了後、営業担当者へメールで通知とPDFリンクを自動送信。担当者はリンクを開いて確認・送付するだけ。

// QuoteFlow — 見積書自動生成コア関数 function generateQuote(e) { const input = e.namedValues; const projectInfo = { clientName: input['顧客名'][0], projectName: input['案件名'][0], memo: input['打ち合わせメモ（自由記述）'][0], budget: input['概算予算'][0], deadline: input['希望納期'][0] }; // Claude APIで見積項目を自動展開 const quoteItems = expandQuoteItems(projectInfo); // スプレッドシートに書き込み writeQuoteToSheet(projectInfo, quoteItems); // PDF化してメール送信 const pdfUrl = exportAsPDF(); sendNotificationEmail(projectInfo.clientName, pdfUrl); } function expandQuoteItems(projectInfo) { const prompt = ` 以下の案件情報から、見積書の項目・数量・単価を生成してください。 案件名: ${projectInfo.projectName} 打ち合わせメモ: ${projectInfo.memo} 概算予算: ${projectInfo.budget}円 JSON形式で返してください: { "items": [ { "category": "カテゴリ", "name": "項目名", "qty": 数量, "unit": "単位", "unit_price": 単価, "note": "備考" } ], "total_estimate": 合計金額, "payment_terms": "支払い条件", "validity_period": "見積有効期限", "cover_message": "見積書の冒頭に入れる挨拶文（200字以内）" }`; const response = UrlFetchApp.fetch('https://api.anthropic.com/v1/messages', { method: 'post', headers: { 'x-api-key': PropertiesService.getScriptProperties().getProperty('CLAUDE_API_KEY'), 'anthropic-version': '2023-06-01', 'content-type': 'application/json' }, payload: JSON.stringify({ model: 'claude-haiku-4-5-20251001', max_tokens: 1200, messages: [{ role: 'user', content: prompt }] }) }); return JSON.parse(JSON.parse(response.getContentText()).content[0].text); }
