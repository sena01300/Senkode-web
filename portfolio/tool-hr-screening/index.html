Architecture / How It Works

処理フロー

📋
Step 01 / Input
Google Forms — 応募者入力

氏名・職務経歴・志望動機・スキル資格を構造化フォームで収集。入力項目はカスタマイズ可能。フォーム送信と同時に処理が自動起動する。

⚙️
Step 02 / Trigger
GAS — フォーム送信トリガー起動

Google Apps Script がフォーム送信イベントを検知。回答データを取得してテキスト整形を実行。採用基準プロンプトと応募者情報を組み合わせる。

🤖
Step 03 / AI Evaluation
Claude API — 自動採点・評価

claude-haiku-4-5 が採用基準に照らして応募者を評価。総合スコア・強み・懸念点・推奨判定をJSON形式で返す。1件あたり平均18秒で処理完了。

📊
Step 04 / Output
Googleスプレッドシート — 結果書き込み

スコア・推奨判定・強み・懸念点・総評をシートに自動記録。採用担当者はシートを開くだけで全応募者を一覧比較できる状態になっている。

✅
Step 05 / Human Decision
採用担当者 — 最終判断

AIの評価はあくまで一次スクリーニング支援。最終的な採否判断は採用担当者が行う。AIが定型作業を肩代わりし、人間はより重要な面接・意思決定に集中できる。

// HireFilter — フォーム送信トリガー関数 function onFormSubmit(e) { const formData = e.namedValues; const applicant = { name: formData['氏名'][0], experience: formData['職務経歴'][0], motivation: formData['志望動機'][0], skills: formData['スキル・資格'][0] }; const score = evaluateApplicant(applicant); writeToSheet(applicant, score); } function evaluateApplicant(applicant) { const prompt = buildPrompt(applicant); const response = UrlFetchApp.fetch('https://api.anthropic.com/v1/messages', { method: 'post', headers: { 'x-api-key': PropertiesService.getScriptProperties().getProperty('CLAUDE_API_KEY'), 'anthropic-version': '2023-06-01', 'content-type': 'application/json' }, payload: JSON.stringify({ model: 'claude-haiku-4-5-20251001', max_tokens: 800, messages: [{ role: 'user', content: prompt }] }) }); return JSON.parse(JSON.parse(response.getContentText()).content[0].text); } function buildPrompt(applicant) { return `以下の応募者情報を採用基準に照らして評価してください。 応募者情報: - 氏名: ${applicant.name} - 職務経歴: ${applicant.experience} - 志望動機: ${applicant.motivation} - スキル・資格: ${applicant.skills} 採用基準: [ここに採用基準を入力] 以下のJSON形式で返してください: { "total_score": 0〜100, "strengths": ["強み1", "強み2"], "concerns": ["懸念点1"], "recommendation": "PASS / HOLD / REJECT", "summary": "100字以内の総評" }`; }

Implementation / 5 Steps

導入手順

01
Google Formsに必要項目を追加

テンプレートを提供。氏名・職務経歴・志望動機・スキルの4項目が基本構成。貴社の採用要件に合わせてカスタマイズ可能。

02
Googleスプレッドシートと連携

Formsの回答先シートを設定するだけ。GASはそのシートに自動でスコア列を追記する。特別な設定は不要。

03
GASスクリプトをコピー&貼り付け

スクリプトエディタを開いてコードを貼り付け。作業時間は5分以内。マニュアルに全手順をスクリーンショット付きで記載。

04
Claude APIキーを設定

スクリプトプロパティにAPIキーを登録。コード内にキーを直書きしないセキュアな実装。Anthropicコンソールで即日発行可能。

05
採用基準を記入してテスト送信

プロンプトの採用基準欄に貴社の選考軸を記入。テスト送信で動作確認。採用基準作成のサポートも提供する。
